### Установка
>note: для текущей версии требуется Node.js версия 18.x или выше
* Склонируйте данный репозиторий
```
git clone https://github.com/PikylNadin/qa-auto-engineer-javascript-project-89.git
```
* Запуск виджет чат-бота:
```
npm run dev
```
